## The Tomo Modules

TomoCooldown is part of a family of standalone addons, each with its own color identity:

| Addon | Color | Purpose |
|---|---|---|
| **TomoPlates** | Violet `#8b5cf6` | Nameplate replacement |
| **TomoFrames** | Amber `#f59e0b` | Unit frame replacement |
| **TomoCooldown** | Cyan `#06b6d4` | Cooldowns, resources, Skyriding |
| **TomoSkins** | Rose `#f43f5e` | UI reskin |
| **TomoCastbar** | Doré `#ffd1b559` | CastBar |

All 5 addons are compatible with each other and can be used in any combination.
